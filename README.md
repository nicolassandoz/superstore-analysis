# 🛒 Superstore Sales Analysis

Analyse exploratoire de données de ventes retail sur le dataset public **Superstore Sales** (Kaggle).

Projet réalisé dans le cadre d'un parcours d'upskilling data analytics — SQL, Python et dataviz.

---

## 🎯 Objectifs

- Explorer et comprendre un dataset de ventes (~10 000 lignes)
- Identifier les produits, régions et périodes les plus rentables
- Produire des visualisations actionnables

---

## 🗂️ Structure du projet
```
superstore-analysis/
├── data/              → dataset source (CSV)
├── sql/               → requêtes d'exploration SQL
├── notebooks/         → analyse Python avec pandas
└── dashboard/         → lien vers le dashboard Looker Studio
```

## 🔧 Outils utilisés

- **SQL** — BigQuery Sandbox (Google)
- **Python** — pandas, matplotlib (via Kaggle Notebooks)
- **Dataviz** — Looker Studio

---

## 📊 Axes d'analyse

- Ventes et profit par catégorie de produit
- Performance par région géographique
- Évolution temporelle des ventes
- Segments clients les plus rentables
